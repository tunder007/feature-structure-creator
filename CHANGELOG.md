## [1.0.0] - 2026-06-22

### Added

- Initial public release.
- Scaffold a feature dossier on a canonical 8-section schema — a dense .md (AI source of truth) plus a generated .html (human view).
- Zero runtime dependencies; requires Node >= 18.
- Hermetic self-test (`npm test`) and CI across Node 18, 20 and 22.
